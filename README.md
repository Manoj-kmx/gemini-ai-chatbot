# Gemini AI Chatbot System

AI-powered chatbot using Google Gemini API and Google Apps Script.

---

## Overview

This project demonstrates how to integrate Google's Gemini AI API to build a chatbot for automated responses.

The system can:

* Generate intelligent replies
* Answer user queries
* Automate conversations
* Be used for lead handling or support

---

## Features

* Gemini API integration
* Dynamic AI responses
* Google Apps Script backend
* Can be extended to CRM or automation workflows

---

## How It Works

1. User sends input
2. Script sends request to Gemini API
3. AI generates response
4. Output is returned

---

## Setup Instructions

### Step 1: Get Gemini API Key

1. Go to Google AI Studio
2. Generate API key

---

### Step 2: Open Apps Script

Create new project in Google Apps Script.

---

### Step 3: Add Script

Paste code from:
/scripts/gemini-chatbot.js

---

### Step 4: Add API Key

Replace:

YOUR_GEMINI_API_KEY

---

### Step 5: Run Function

Run:
generateResponse()
